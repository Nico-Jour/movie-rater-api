## Description

Movie-Rater-Api is a project that let's you vote for your favorite movie and keep track of all the votes and the most voted one.

## Installation

```bash
$ npm install
```
and you need docker installed in your machine

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
